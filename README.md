# 💠 Pomodoro Timer

React + Vite + Tailwind CSS で構築したポモドーロ・テクニック用タイマーアプリです。

## ポモドーロ・テクニックとは

25分の集中作業と短い休憩を繰り返す時間管理術です。
4セッションごとに長めの休憩を挟むことで、集中力を維持しながら疲労を軽減します。

| フェーズ | 時間 |
|----------|------|
| 作業 | 25分 |
| 小休憩 | 5分 |
| 長休憩 | 15分 または 30分（切り替え可） |

---

## 機能一覧

- **円形プログレスバー** — 残り時間をSVGの円弧でリアルタイム表示
- **フェーズ自動遷移** — 作業→休憩→作業を自動で切り替え
- **ブラウザ通知** — フェーズ終了時にデスクトップ通知を送信
- **効果音** — Web Audio API による作業完了・休憩完了の通知音
- **セッションドット** — 現在サイクル内の進捗を4つのドットで表示
- **日付・現在時刻** — 秒単位でリアルタイム表示
- **長休憩時間の切り替え** — 15分 / 30分 をワンクリックで変更
- **カラーテーマ選択** — オーシャン / ミッドナイト / アークティック の3種類から選択
- **波紋エフェクト** — タイマー稼働中にランダムな位置・サイズの波紋がアニメーション表示
- **ダーク / ライトモード** — ボタン1つで即切り替え
- **ブラウザタブタイトル更新** — バックグラウンドでも残り時間を確認可能

### 目標ポモドーロ設定モード

ポモドーロ数（1〜20）を指定して計画的にタイマーを実行できます。

- **終了予定時刻**（年月日・曜日・時刻）を事前および実行中にリアルタイム表示
- **所要時間**を X時間Y分 形式で表示
- **計画開始**ボタンで作業・休憩を全自動で進行
- **スケジュールバー** — 作業／小休憩／長休憩の構成を積み上げ棒グラフで可視化（選択テーマの色に連動）
- **▼ マーカー** — 現在位置をリアルタイムでスライド表示
- **進捗バー** — 完了セッション数を進捗バーで表示
- **一時停止 / キャンセル** にも対応

## セットアップ

### 必要環境

- Node.js 18 以上
- npm 9 以上

### インストール

```bash
cd Pomodoro_Timer
npm install
```

---

## 起動方法

### 開発サーバー（ホットリロードあり）

```bash
npm run dev
```

起動後、ブラウザで以下にアクセスしてください：

```
http://localhost:5173
```

### 本番ビルド

```bash
npm run build
```

`dist/` フォルダに静的ファイルが生成されます。

### ビルドのプレビュー

```bash
npm run preview
```

ビルド後のファイルをローカルでプレビューします（デフォルト: `http://localhost:4173`）。

## ブラウザ通知の許可

初回アクセス時にブラウザから通知の許可を求めるダイアログが表示されます。
「許可」を選択することでフェーズ終了時にデスクトップ通知が届きます。

> 許可しなくても、効果音とUI上の表示でフェーズ終了を確認できます。

## 使い方

### 基本操作

1. **▶ ボタン** — タイマー開始
2. **⏸ ボタン** — 一時停止（再度押すと再開）
3. **↺ ボタン** — 現在フェーズをリセット
4. **⏭ ボタン** — 次のフェーズへスキップ
5. フェーズタブ（作業中 / 小休憩 / 長休憩）で手動切り替え可能
6. ヘッダーの **3つの色丸** でカラーテーマを切り替え（オーシャン / ミッドナイト / アークティック）
7. **長休憩の時間** トグルで 15分 / 30分 を選択

### 計画モード

1. 画面下部の「目標ポモドーロ数を設定して開始」セクションで **− / ＋** ボタンを押して数を選択
2. 終了予定時刻と所要時間を確認
3. **「この計画で開始」** を押すと自動進行スタート
4. スケジュールバーの ▼ マーカーで現在位置を確認
5. **「キャンセル」** で手動モードに戻る
